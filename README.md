Landing page oficial da Academia M2 Move, desenvolvida para apresentar a infraestrutura completa, modalidades esportivas e canais de contato da academia. O projeto conta com menu de navegação fixo, seção de benefícios, carrossel de imagens automático, mapa interativo e links diretos para redes sociais.


## 📋 Sobre o Projeto
A Landing Page da M2 Move tem como objetivo:

Apresentar as modalidades oferecidas (Natação, Muay Thai, Hidroginástica, Musculação)

Destacar os benefícios de cada atividade

Mostrar a infraestrutura através de um carrossel de imagens

Fornecer localização via Google Maps

Facilitar o contato através de WhatsApp, Instagram e telefones

Incentivar a matrícula com CTA direto para o WhatsApp

## 🚀 Funcionalidades Implementadas
✅ Navegação
Menu fixo com links âncora para todas as seções

Logo da academia no topo

Botão "Matricule-se" com link direto para WhatsApp

✅ Seção de Modalidades
Cards com 4 atividades:

🏊 Natação - 6 benefícios listados

🥊 Muay Thai - 5 benefícios listados

💧 Hidroginástica - 5 benefícios listados

💪 Musculação - 5 benefícios listados

✅ Carrossel de Imagens (Slider)
5 imagens da infraestrutura

Navegação manual por botões

Transição automática (funcionalidade via CSS/JS - a ser verificada)

Fundo na cor rgb(2, 2, 45)

✅ Seção de Contato e Localização
3 números de telefone (Unidades 1, 2 e 3)

Ícones do Instagram e WhatsApp com links externos

Mapa do Google Maps integrado (busca por "m2move")

Rodapé com logo e direitos reservados

## 🛠️ Tecnologias Utilizadas

HTML5	

CSS3

JavaScript	

Google Maps Embed API
